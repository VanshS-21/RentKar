# RentKar Product Overview

RentKar is a peer-to-peer item sharing platform designed for college students to borrow and lend items within their campus community.

## Core Purpose
Enable students to share items temporarily instead of purchasing them, making campus life more affordable and sustainable.

## Target Users
College students who need short-term access to items like books, calculators, electronics, and accessories.

## Key Features
- User authentication and profiles
- Item listing with image upload
- AI-powered item description generation (Google Gemini)
- Borrow request workflow (request → approve/reject → return → complete)
- Item status tracking (Available, Borrowed, Unavailable)
- Optional admin controls for content moderation

## Business Rules
- Only authenticated users can list items or make borrow requests
- Item owners (lenders) approve or reject incoming requests
- Items can only be borrowed when status is "Available"
- Borrowers and lenders can track request status through the workflow
- All transactions are peer-to-peer; platform facilitates connections only

## Project Context
Academic project for PW IOI – School of Technology, 10-12 week timeline, 3-person team.
